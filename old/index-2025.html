Patrick's Props
Which team will have the better line score poker hand?


Pushable if both teams' line score poker hands are exactly the same, including kickers where relevant.

line-score poker hand
At the end of the game, each team will have a line score comprised of the number of points they scored in the first quarter, then the second quarter, third quarter, fourth quarter, any overtime periods, and finally their final score. Using Super Bowl LIV as an example, Kansas City's line score was "7 3 0 21 31" and San Francisco's was "3 7 10 0 20". Using the digits of the line score, you create the best five-card poker hand possible. In Kansas City's case, that would be two pair: "3 3 1 1 7"; in San Francisco's case, it's three-of-a-kind: "0 0 0 7 3". Three-of-a-kind is better than two pair so San Francisco would have had the superior hand.
What will be the sum of the digits that comprise the final score?


For example, a game ending 21-17 results in a sum-of-digits of 2 + 1 + 1 + 7 = 11.

Fibonacci number
For F(0) = 0, F(1) = 1, and F(n) = F(n - 1) + F(n - 2), a Fibonacci number is a non-negative integer x such that x = F(n) for some non-negative integer value of n. The first ten Fibonacci numbers in sequence are: 0, 1, 1, 2, 3, 5, 8, 13, 21, and 34.
prime number
A prime number is a positive integer x greater than 1 whose only positive integer factors are 1 and itself. The first ten prime numbers are 2, 3, 5, 7, 11, 13, 17, 19, 23, and 29.
